# Redis代码注释

Reids版本：3.2.11

本注释版本不会删除原始代码中的任何东西，只会增加注释。本注释版本的阅读顺序参考了[如何阅读Redis源码](https://github.com/huangz1990/blog/blob/master/diary/2014/how-to-read-redis-source-code.rst)。

本注释项目只是个人爱好，受技术水平和时间限制，出差错在所难免，有感兴趣的同学欢迎指出问题、交流和提交issue。
